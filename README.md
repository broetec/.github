# Broetec `.github`

This repository holds **organization-level defaults** for [Broetec](https://github.com/broetec) on GitHub: community guidelines, automation, and the text shown on the organization profile.

## What lives here

| Location | Purpose |
| -------- | ------- |
| [`profile/README.md`](profile/README.md) | Public **organization profile** README (what visitors see on the org page). |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Global contributing guide for Broetec projects. |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Contributor Covenant code of conduct. |
| [`SECURITY.md`](SECURITY.md) | How to report security vulnerabilities. |
| [`LICENSE`](LICENSE) | Default license reference where applicable. |
| [`.github/`](.github/) | Issue and PR templates, label config, and workflows. |
| [`templates/`](templates/) | Shared templates (for example Makefile snippets for Python projects). |

## Contributing

See [`CONTRIBUTING.md`](CONTRIBUTING.md). For questions that are not tied to a specific repository, use the contact options in [`profile/README.md`](profile/README.md).
